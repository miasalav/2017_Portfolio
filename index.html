Fashion Group International of Toronto

View Website

HTML/CSS/SASS/WORDPRESS/PHP/JSON/CURL

This my final RED Academy project. I worked on developing this website with my project partner, Caitlin Dyk. This was a collaboration between the UX Design team at RED Academy who researched and designed the website and passed off the design files to our dev team.

I mostly worked on grabbing information from Facebook, Twitter, and Instagram and tweaking a plugin that automatically creates a WP post of every social media post from the organization's three social media accounts. We created the website through tweaking the Understrap starter template. We made every element on this template dynamic so the client can change, add, or delete any element on the website they wish to manipulate. This website is mobile friendly.
